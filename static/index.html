Customer Relation Management API

Get a customer

Endpoint: /customers/{id}

Method: GET

Description: Retrieves a single customer infomation by their id

Get all customers

Endpoint: /customers

Method: GET

Description: Retrieves all customers

Create a customer

Endpoint: /customers

Method: POST

Description: Create a new customer

Update a customer

Endpoint: /customers/{id}

Method: PUT

Description: Update an existing customer by their id

Delete a customer

Endpoint: /customers/{id}

Method: DELETE

Description: Delete an existing customer by their id
